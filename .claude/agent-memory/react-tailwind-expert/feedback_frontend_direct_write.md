The frontend can be scaffolded/written directly to files (npm install, generate all
files, build). Only the **backend** is the part the owner copies by hand to learn.

**Why:** The owner is learning the backend specifically; the frontend is a tool to
exercise it, not the learning target.

**How to apply:** For frontend work, just write the files and verify the build/dev
server. Still provide a concise summary of structure and key files at the end so the
owner understands what was created. See [[user-profile]].
