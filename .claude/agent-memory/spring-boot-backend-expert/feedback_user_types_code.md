For CODE changes, describe what to change and why — do NOT apply edits automatically. The user types code by hand to learn from it.

**Why:** Stated in the project PRD as the user's working method; this is a learning project.

**How to apply:** Explain the change with a snippet and location, then let the user implement. Exception: concrete non-code chores explicitly requested (e.g. moving image files) — go ahead and do those directly. The user is a Spring Boot learner.
